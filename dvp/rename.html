📌 How it works
• {0:D3} creates 3-digit numbers: 001, 002... 100
• Files renamed in alphabetical order
• Always preview before actual rename!
⚡ Commands
📁 NAVIGATE cd command
cd "C:\Users\User\Documents\Cuba"

✅ ACTUAL RENAME ⚠️ Permanent
