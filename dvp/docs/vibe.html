🎙️ Vibe · Transcribe on your own

offline · privacy-first Local audio/video transcription using OpenAI Whisper — no cloud, no uploads, full control.

Vibe is a cross-platform desktop app (Windows, macOS, Linux) that turns speech into text entirely on your machine. It supports batch processing, microphone input, YouTube links, speaker diarization, and exports to SRT, PDF, DOCX, JSON, and more.

💾 ~87 MB (app) + model 🧠 1.5 GB (Large v3) · 769 MB (Medium) · 244 MB (Small)

📥 1. Download & install

  • Visit the official releases page on GitHub (6.7k ⭐, 72 releases).
  • Download the installer for your system: .exe (Windows), .dmg (macOS), .AppImage or .deb (Linux).
  • Run the installer – it takes about 87 MB on disk.
  • On first launch, Vibe will automatically download the Large v3 model (~1.5 GB) — you can change it later (see step 3).

⚙️ 2. First run – load your audio

You have three ways to bring audio in:

  • 📁 Local file – click Select File and choose MP3, WAV, M4A, FLAC, MP4, etc.
  • 🔗 Online link – switch to the clip/chain tab, paste a YouTube / Vimeo / Facebook URL, then click Download and transcribe.
  • 🎤 Microphone – go to the mic tab, select your input device, and record live (realtime transcription).

🧠 3. Choose your model – speed vs accuracy

Vibe lets you switch models anytime in Settings → Available Models. Here’s the real‑world trade‑off:

⚡ Tiny 🚀 fastest
~39 MB · low accuracy, for quick tests
⚡ Small 🚀 very fast
~244 MB · good accuracy, good speed
⭐ Medium ⚡ balanced
~769 MB · excellent accuracy · recommended
🔥 Large v3 🐢 slower
~1.5 GB · highest accuracy, slower
🚀 Large v3 Turbo ⚡ faster than v3
~1.5 GB · near‑highest accuracy, quicker

💡 tip Start with Medium — it’s the sweet spot for most users. If you need speed, switch to Small; if you need max precision, use Large v3 Turbo.

🧩 4. Extra tweaks before transcribing

  • Language – choose the spoken language (auto‑detect works, but manual selection improves accuracy).
  • Speaker diarization – enable Recognise speakers to distinguish different voices.
  • Stable timestamps – use for subtitle‑grade timing (VAD‑backed, but slower).
  • Summarize – connect to Ollama or Claude API for automatic summaries.

▶️ 5. Run & export

  1. Click the big Transcribe button – the job will appear in the queue.
  2. Once done, you can copy the text or export in multiple formats: TXT, SRT, VTT, PDF, DOCX, HTML, JSON.
  3. Use the print icon to send the transcript directly to your printer.

🔧 6. Power features (optional)

  • Batch processing – select a whole folder and transcribe all files at once.
  • CLI mode – launch Vibe from the terminal with vibe --help to see all arguments.
  • HTTP API – run with --server and open http://localhost:3022/docs for Swagger documentation.

🗑️ 7. Uninstall Vibe completely

Removing Vibe and all its components (models, cache, settings) is straightforward. Follow the steps for your operating system:

🪟 Windows

  • Via Settings: go to Settings → Apps → Installed apps, find Vibe, click the three dots and select Uninstall.
  • Via Control Panel: open Programs and Features, find Vibe, right‑click and select Uninstall.
  • Manual cleanup (to remove all leftover data):
    • Delete the folder %APPDATA%\Vibe (contains models, settings, and cache).
    • Delete %LOCALAPPDATA%\Programs\Vibe if it remains.

🍎 macOS

  • Open Finder → Applications, drag the Vibe.app to the Trash, then empty the Trash.
  • Remove user data – delete the folder ~/Library/Application Support/Vibe (models, preferences).
  • Also check ~/Library/Caches/Vibe and ~/Library/Preferences/com.vibe.app.plist – you can delete these as well.

🐧 Linux

  • If installed via .deb: run sudo apt remove vibe (or sudo dpkg -r vibe).
  • If using AppImage: simply delete the .AppImage file.
  • Remove config & models: delete the folder ~/.config/Vibe (or ~/.local/share/Vibe depending on your distribution).
💡 Tip: To free up the most space, always delete the models folder (usually inside the app data directory). Models can take up to 1.5 GB each.

Vibe — это кроссплатформенное приложение (Windows, macOS, Linux), которое превращает речь в текст прямо на вашем компьютере. Поддерживает пакетную обработку, запись с микрофона, ссылки YouTube, разделение дикторов и экспорт в SRT, PDF, DOCX, JSON и другие форматы.

📥 1. Скачать и установить

  • Скачайте установщик для вашей системы: .exe (Windows), .dmg (macOS), .AppImage или .deb (Linux).
  • Запустите установку — приложение займёт около 87 МБ на диске.
  • При первом запуске Vibe автоматически загрузит модель Large v3 (~1.5 ГБ) — вы сможете сменить её позже (см. шаг 3).

⚙️ 2. Первый запуск – загрузка аудио

Есть три способа загрузить аудио:

  • 📁 Локальный файл – нажмите Select File и выберите MP3, WAV, M4A, FLAC, MP4 и т.д.

  • 🎤 Микрофон – перейдите на вкладку микрофона, выберите устройство и записывайте в реальном времени.

🧠 3. Выбор модели – скорость и точность

~39 МБ · низкая точность, для быстрых тестов

💡 совет Начните с Medium — это золотая середина для большинства пользователей. Если нужна скорость — ставьте Small, а если максимальная точность — Large v3 Turbo.

🧩 4. Дополнительные настройки перед транскрипцией

  • Язык – выберите язык речи (автоопределение работает, но ручной выбор повышает точность).
  • Распознавание дикторов – включите Recognise speakers, чтобы различать голоса.
  • Стабильные метки времени – для субтитров с высокой точностью (медленнее).
  • Суммаризация – подключите Ollama или Claude API для автократкого содержания.

▶️ 5. Запуск и экспорт

  1. Нажмите большую кнопку Transcribe – задача появится в очереди.
  2. После завершения вы можете скопировать текст или экспортировать в форматах: TXT, SRT, VTT, PDF, DOCX, HTML, JSON.
  3. Используйте иконку печати, чтобы отправить транскрипт на принтер.

🔧 6. Продвинутые функции (опционально)

  • Пакетная обработка – выберите целую папку и транскрибируйте все файлы сразу.
  • Режим командной строки (CLI) – запустите Vibe из терминала с vibe --help, чтобы увидеть все аргументы.
  • HTTP API – запустите с --server и откройте http://localhost:3022/docs для документации Swagger.

🗑️ 7. Полное удаление Vibe

Удалить Vibe и все его компоненты (модели, кэш, настройки) очень просто. Следуйте инструкциям для вашей операционной системы:

🪟 Windows

  • Через настройки: перейдите в Параметры → Приложения → Установленные приложения, найдите Vibe, нажмите три точки и выберите Удалить.
  • Через панель управления: откройте Программы и компоненты, найдите Vibe, нажмите правой кнопкой и выберите Удалить.
  • Ручная очистка (чтобы удалить все остаточные данные):
    • Удалите папку %APPDATA%\Vibe (там хранятся модели, настройки и кэш).
    • Удалите %LOCALAPPDATA%\Programs\Vibe, если она осталась.

🍎 macOS

  • Откройте Finder → Программы, перетащите Vibe.app в Корзину, затем очистите Корзину.
  • Удаление пользовательских данных – удалите папку ~/Library/Application Support/Vibe (модели, настройки).
  • Также проверьте ~/Library/Caches/Vibe и ~/Library/Preferences/com.vibe.app.plist – их тоже можно удалить.

🐧 Linux

  • Если установлено через .deb: выполните sudo apt remove vibe (или sudo dpkg -r vibe).
  • Если использовался AppImage: просто удалите файл .AppImage.
  • Удаление конфигурации и моделей: удалите папку ~/.config/Vibe (или ~/.local/share/Vibe в зависимости от дистрибутива).
💡 Совет: Чтобы освободить больше всего места, всегда удаляйте папку с моделями (обычно внутри каталога с данными приложения). Модели могут занимать до 1.5 ГБ каждая.
