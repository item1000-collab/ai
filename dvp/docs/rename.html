🔢 HOW 001, 002, 003 NUMBERING WORKS
PowerShell uses the format operator {0:D3}
• D3 = Decimal with 3 digits (padding zeros)
• 1 becomes 001 → 10 becomes 010 → 100 becomes 100
• Script loops through files alphabetically and assigns incrementing index.
• Example: photo.jpg → 001.jpg, doc.pdf → 002.pdf, etc.
• Perfect for albums, archives, and sorted lists!
✂️ REMOVE FIRST N CHARACTERS (e.g., first 4 chars)
Command: $_.Name.Substring(4)
• .Substring(4) = removes characters at positions 0,1,2,3 → keeps from index 4 onward.
• Example: "IMG_1234.png" → becomes "1234.png" (removes "IMG_")
• Example: "VID_2024.mp4" → "2024.mp4"
• ⚠️ Safe preview: Use -WhatIf flag to see changes without renaming.
📌 PowerShell Essentials
✅ Preview before rename: Always test with -WhatIf or write-host previews.
✅ Ordering: Files are renamed in default alphabetical order (by name).
✅ Extensions preserved automatically via $_.Extension.
