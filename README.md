# AI Exploration Hub · Anatoly's Site

A curated, clean, and interactive HTML/CSS dashboard for exploring AI tools, experimental labs, and community feedback.  
The site is designed as a personal knowledge hub — combining useful AI resources, in‑development projects, and a working comment system.

👉 **[View live site](https://item1000-collab.github.io/ai/)** (replace with your actual GitHub Pages URL)

---

## ✨ Features

### 📚 AI Tools & Resources
Quick‑access grid of popular AI playgrounds:
- DeepSeek Chat, Qwen LM, Sesame, Google AI Studio, Grok, ChatGPT, NotebookLM, Character.AI, LM Arena, Vheer Dashboard.

### 🛠️ Developments (Applied AI Experiments)
- **Visual Style Engine** – real‑time style lab with 1000+ styles.  
- **NotebookLM Style Guide (PDF)** – downloadable visual reference for presentations.  
- *GitHub Link Builder* – convert filename to raw GitHub Pages URL · copy instantly.

### 💬 Feedback & Comments
- Fully client‑side comment system using `localStorage`.  
- Add, delete, and sort comments (newest first).  
- No backend required – perfect for static hosting (GitHub Pages, etc.).

### 📱 Responsive Design
- Works on desktop, tablet, and mobile.  
- Clean card layout with subtle animations and hover effects.

---

## 🛠️ Tech Stack

- **HTML5** + **CSS3** (flexbox, grid, modern styling)  
- **Vanilla JavaScript** (no external dependencies)  
- **LocalStorage** for comment persistence  
- **Google Fonts / System UI** typography
